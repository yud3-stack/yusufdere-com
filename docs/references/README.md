# Homepage Visual Reference

Reference file:

docs/references/homepage-design-reference.png

Use this image only as visual direction for the homepage layout, spacing and mood.

## Key layout notes

- Dark premium personal website layout.
- Top navigation is minimal and quiet.
- Hero section uses a split layout.
- Left side contains identity:
  - YD mark
  - Yusuf Dere
  - Building the life I imagined.
  - location
  - small CTA
- Right side contains a large cinematic image area.
- Below the hero, sections are arranged with generous spacing.
- About section feels editorial, not CV-like.
- Project cards are minimal and subtle.
- Now and Journal can share a two-column layout.
- Gallery and Uses can share a two-column layout.
- Footer is calm and minimal.

## What to avoid

- Do not copy the design 1:1.
- Do not create a generic SaaS landing page.
- Do not make it colorful.
- Do not use excessive gradients.
- Do not use the personal reference images in public UI.
- Do not make it look like a resume website.
