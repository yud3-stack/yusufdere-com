# Design Rules

## Visual Direction

The website must feel premium, minimal, modern and timeless.

Dark theme is the primary design direction.
Light theme should be added later only after the dark theme feels complete.

## Layout

Use:
- large typography
- generous spacing
- strong grid alignment
- subtle section dividers
- cinematic image placement
- clean navigation
- calm card layouts

Avoid:
- crowded sections
- too many CTAs
- colorful SaaS landing page style
- heavy gradients
- overused glassmorphism
- excessive shadows
- childish icons

## Homepage Reference

The homepage should follow a split hero design.

Left panel:
- YD logo
- Yusuf Dere
- Building the life I imagined.
- Samsun, Türkiye
- About me button

Right panel:
- large cinematic personal hero image

Below:
- About
- Projects
- Now
- Journal
- Gallery
- Uses
- Contact

## Colors

Dark theme first:

- Background: near black
- Surface: dark charcoal
- Border: low opacity white
- Text primary: near white
- Text secondary: soft gray
- Accent: minimal, neutral, not colorful

Avoid strong purple/blue gradients except very subtle project-related accents.

## Typography

Use large, clean typography.

Headings should feel editorial and premium.
Body text should be readable and calm.

Recommended font strategy:
- Use Geist or Inter first.
- Avoid decorative fonts.

## Motion

Motion should be subtle and elegant.

Use:
- fade in
- soft upward reveal
- hover opacity
- slight card lift

Avoid:
- bounce
- rotation
- heavy parallax
- aggressive scale effects

## Responsive

Mobile must not feel like an afterthought.

Hero should stack cleanly:
- identity first
- image second
- sections below

Navigation should stay minimal.
