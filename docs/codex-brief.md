# Codex Brief

You are working on YusufDere.com.

Your task is to build a premium personal brand website using Next.js, TypeScript, Tailwind CSS, Framer Motion and Sanity CMS.

Do not build a generic portfolio template.
Do not make it look like a CV.
Do not make it look corporate.

The design should follow a dark, cinematic, minimal, premium direction.

## First Implementation Goal

Build the foundation first.

Priority order:
1. Project structure
2. Global design system
3. Layout shell
4. Theme setup
5. Sanity client setup
6. CMS schema setup
7. Static homepage using placeholder data
8. Then connect CMS data

## Required Components

Create reusable components:

- SiteHeader
- SiteFooter
- ThemeToggle
- Container
- SectionHeader
- ProjectCard
- JournalItem
- GalleryPreview
- UsesGrid
- NowList
- Button
- Badge

## Homepage Sections

Implement:
- HeroSection
- AboutSection
- ProjectsSection
- NowJournalSection
- GalleryUsesSection
- ContactSection

## Technical Requirements

- Use App Router
- Use TypeScript
- Use Tailwind CSS
- Use next/image
- Use Lucide icons
- Use Framer Motion only where necessary
- Keep components organized
- Keep data typed
- Keep CMS queries separate
- Use semantic HTML
- Make it responsive

## Do Not

- Do not add shadcn/ui unless explicitly requested.
- Do not add Material UI.
- Do not add Bootstrap.
- Do not overuse animations.
- Do not hardcode all content inside page.tsx.
- Do not create messy components.
- Do not ignore mobile layout.
