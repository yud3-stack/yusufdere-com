# Content Checklist

This checklist tracks Turkish CMS fields that should be filled in Sanity Studio so `/tr` feels intentionally localized. English fields should remain filled for the default site.

## Site Settings

- `bioTr`
- `seoTitleTr`
- `seoDescriptionTr`
- `locationTr`

## About

- `titleTr`
- `eyebrowTr`
- `introTr`
- `bodyTr`
- `currentFocusTr`
- `focusAreasTr`

## Projects

For each project:

- `titleTr`
- `shortDescriptionTr`
- `descriptionTr`
- `categoryLabelTr`

Shared fields that do not need translation:

- `slug`
- `status`
- `category`
- `techStack`
- `liveUrl`
- `githubUrl`

## Journal

For each post:

- `titleTr`
- `excerptTr`
- `contentTr`

Shared fields that do not need translation:

- `slug`
- `date`
- `category`
- `published`
- `featured`

## Gallery

For each image:

- `titleTr`
- `descriptionTr`
- `locationTr`

Shared fields that do not need translation:

- `image`
- `category`
- `date`
- `featured`
- `order`

## Uses

For each item:

- `titleTr`
- `descriptionTr`
- `categoryLabelTr`

Shared fields that do not need translation:

- `category`
- `icon`
- `url`
- `order`
- `featured`

## Now

For each item:

- `titleTr`
- `descriptionTr`

Shared fields that do not need translation:

- `icon`
- `active`
- `order`
